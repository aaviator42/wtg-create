=====================

Follow these steps!
-------------------
1a. Rename your Windows iso file to "win.iso"
1b. [OR] Extract "install.wim" from the iso file.
1c. [OR] Rename your modified .wim file to "install.wim"

2. Place the iso/wim file, and and all files that came with this script in the same directory.

3. Run this script as administrator

4. Follow the instructions.

NOTE:
1. This program requires 7-zip (7z.exe) and ImageX (imagex.exe) to extract files.
2. No disk with allocated disk letter "R" should exist.
